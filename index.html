AI Instructions

mAInifold — AI-driven browser CAD. Coordinate system: right-handed, Z-up. Full docs: /ai.md

Console API — window.mainifold

mainifold.run(code?)        // Run code, returns geometry stats
mainifold.validate(code)    // Check without render → {valid, error?}
mainifold.getGeometryData() // Current stats {status, volume, ...}
mainifold.getCode()         // Read editor
mainifold.setCode(code)     // Set editor (no auto-run)
mainifold.sliceAtZ(z)       // Cross-section → polygons + area
mainifold.getBoundingBox()  // → {min:[x,y,z], max:[x,y,z]}
mainifold.toggleClip(on?)   // 3D clip plane toggle
mainifold.setClipZ(z)       // Set clip height
mainifold.exportGLB()       // Download GLB
mainifold.exportSTL()       // Download STL
mainifold.exportOBJ()       // Download OBJ
mainifold.export3MF()       // Download 3MF
// Isolated execution & assertions
await mainifold.runIsolated(code)  // Test without side effects
await mainifold.runAndAssert(code, {minVolume, isManifold, ...})
mainifold.isRunning()       // Check execution state
// Sessions
await mainifold.createSession(name?)
await mainifold.runAndSave(code, label?)  // Returns stat diff
await mainifold.createSessionWithVersions(name, [{code,label},...])
mainifold.getGalleryUrl()   // Gallery for human review

Code pattern — must return a Manifold. All transforms are immutable.

const { Manifold, CrossSection } = api;
const box = Manifold.cube([10,10,10], true);
const hole = Manifold.cylinder(12, 3, 3, 32);
return box.subtract(hole);

Primitive origins

cube([x,y,z])       → [0,0,0] to [x,y,z]. center=true → centered
sphere(r, n?)       → centered at origin
cylinder(h,rLo,rHi?,n?) → Z-axis, base z=0, top z=h. rHi=0 for cone
tetrahedron()       → ~2-unit bbox at origin. Scale to size.
extrude(cs, h, nDiv?, twist?, scaleTop?, center?)
                    → along Z. twist=degrees, scaleTop=0 for cone point
revolve(cs, n?, deg?) → around Y axis, result is Z-up automatically.
                    Profile X=radial, Y=height → Y becomes Z.

Constructors: cube, sphere, cylinder, tetrahedron, extrude, revolve, union, difference, intersection, hull, compose, smooth, levelSet, ofMesh

Transforms: translate, rotate (degrees), scale, mirror, warp, transform

Booleans: add, subtract, intersect, hull

Queries: volume, surfaceArea, genus, numVert, numTri, isEmpty, boundingBox, status (0=valid), slice, project, getMesh, decompose

CrossSection: square, circle, ofPolygons (CCW outer, CW holes), hull, compose, union, difference, intersection

#geometry-data JSON includes: status, vertexCount, triangleCount, boundingBox (with dimensions), centroid, volume, surfaceArea, genus, isManifold, componentCount, crossSections (z25/z50/z75), executionTimeMs. On error: status:"error" + error message.

Tip: Add ?view=ai for 4 isometric views on load.
